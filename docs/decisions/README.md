# Architecture Decision Records

This directory holds Tequity's organization-level ADRs — the durable record of
architecturally significant, hard-to-reverse decisions. Each decision is its own
directory, `NNNN-kebab-title/README.md`, with the next zero-padded number.

An ADR captures a *decision* (durable — accepted, superseded, never edited to
reverse); a tracker issue captures *work* (transient — open → closed). Argue the
decision in the issue/PR, then distill it here.
